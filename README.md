## Project

"Adventure-style game, which player has escaped from a group of bandits! He must now avoid being eliminated and make it back to the safe house."

## How to Run
Clone Repo then Unzip LEAF. Double Hack2020 executable.

## Controls
A         -> go left

D         -> go right

spacebar  -> jump

mouse     -> aim

L-click   -> shoot arrow
